Jeu de données les plus récentes

Voici le jeu de données contenant seulement l’observation la plus récente pour chacune des juridictions (pays ou continent). Cette version du jeu de données contient 191 observations et 24 variables.

Ce jeu de donnée est également augmenté avec la variable qui identifie le groupe (cluster) auquel chaque pays appartient.

Jeu de données complet

Étant très volumineux, le jeu de données complet qui contient 65974 observations et 24 variables est disponible sur demande seulement. La particularité de ce jeu de données est qu’il permet de suivre quotidiennement l’évolution des campagnes de vaccination partout dans le monde.
